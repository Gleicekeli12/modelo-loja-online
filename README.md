# 🛍️ Modelo Loja Online

Site desenvolvido para simular uma loja virtual de produtos naturais, criado por **Gleice Keli Felipe** com foco em prática de **HTML**, **CSS** e **JavaScript**, além de integração com dados externos.

## 🌐 Descrição

Projeto de loja online moderna, leve e responsiva, ideal para pequenos negócios que desejam vender pela internet de forma simples.

O sistema permite visualizar produtos, acessar detalhes e realizar pedidos diretamente via WhatsApp, oferecendo uma experiência prática tanto para o cliente quanto para o vendedor.

## 🚀 Funcionalidades

🛒 Exibição de produtos em formato de catálogo

🔍 Barra de pesquisa para encontrar produtos

📂 Filtro por categorias

📄 Página individual de cada produto

📱 Compra facilitada via WhatsApp

🌐 Layout totalmente responsivo

🔗 Integração com Google Sheets (via CSV)

## 🚀 Como Rodar Localmente

Clone o repositório:

git clone https://github.com/gleicekeli12/modelo-loja-online.git

Acesse a pasta do projeto:

cd nome-do-projeto

Abra o arquivo index.html no navegador.

Você pode clicar duas vezes no arquivo ou usar Live Server no VS Code para uma melhor experiência.
